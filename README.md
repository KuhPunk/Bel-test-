# Беларуская Mini App - Система регистрации

## Запуск приложения

Приложение уже запущено на http://localhost:8000

## Функционал системы регистрации

### Роли пользователей:

1. **Ученик (Student)**
   - Регистрация: Имя, Фамилия, Номер группы
   - Может проходить тесты
   - Результаты сохраняются в профиле
   - Не требуется пароль для входа

2. **Учитель (Teacher)**
   - Регистрация: Имя пользователя, Пароль
   - Может создавать тесты и правила
   - Видит результаты всех учеников по группам
   - Требуется вход с паролем

3. **Администратор (Admin)**
   - Вход через секретный ключ: `belarus_admin_2024`
   - Полный доступ к системе
   - Может просматривать всех пользователей
   - Может удалять пользователей
   - Видит пароли учителей

## Как протестировать:

### Регистрация ученика:
1. Откройте http://localhost:8000
2. Перейдите на вкладку "Регистрация"
3. Выберите роль "Ученик"
4. Заполните: Имя, Фамилия, Номер группы
5. Нажмите "Зарегистрироваться"

### Регистрация учителя:
1. Откройте http://localhost:8000
2. Перейдите на вкладку "Регистрация"
3. Выберите роль "Учитель"
4. Заполните: Имя пользователя, Пароль
5. Нажмите "Зарегистрироваться"

### Вход администратора:
1. Откройте http://localhost:8000
2. Нажмите "Вход для администратора"
3. Введите секретный ключ: `belarus_admin_2024`
4. Нажмите "Войти"

## Сохранение данных

Все данные сохраняются в localStorage браузера:
- Пользователи
- Результаты тестов
- Правила
- Достижения
