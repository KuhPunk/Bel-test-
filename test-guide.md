# Руководство по тестированию системы регистрации

## Сценарий 1: Регистрация и вход ученика

1. Откройте http://localhost:8000
2. Вы увидите экран входа с двумя вкладками: "Вход" и "Регистрация"
3. Перейдите на вкладку "Регистрация"
4. В выпадающем списке "Роль" выберите "Ученик"
5. Появятся поля: Имя, Фамилия, Номер группы
6. Заполните:
   - Имя: Іван
   - Фамилия: Петраў
   - Номер группы: 101
7. Нажмите "Зарегистрироваться"
8. Вы автоматически войдёте в систему
9. Пройдите тест и проверьте, что результат сохранился

## Сценарий 2: Регистрация учителя

1. Откройте http://localhost:8000 в режиме инкогнито (или очистите localStorage)
2. Перейдите на вкладку "Регистрация"
3. Выберите роль "Учитель"
4. Появятся поля: Имя пользователя, Пароль
5. Заполните:
   - Имя пользователя: teacher1
   - Пароль: password123
6. Нажмите "Зарегистрироваться"
7. Вы войдёте в систему как учитель
8. Создайте тест и проверьте функционал

## Сценарий 3: Вход учителя

1. Выйдите из системы (кнопка "Выйти")
2. На экране входа введите:
   - Имя пользователя: teacher1
   - Пароль: password123
3. Нажмите "Войти"
4. Вы должны войти в систему

## Сценарий 4: Вход администратора

1. Откройте http://localhost:8000
2. Нажмите "Вход для администратора"
3. В модальном окне введите секретный ключ: belarus_admin_2024
4. Нажмите "Войти"
5. Вы увидите панель администратора с:
   - Списком всех учителей (с паролями)
   - Списком всех учеников (с результатами)
   - Кнопками для удаления пользователей

## Сценарий 5: Просмотр результатов (учитель)

1. Войдите как учитель
2. Зарегистрируйте несколько учеников (в разных окнах браузера)
3. Пусть ученики пройдут тесты
4. Войдите как учитель
5. Вы увидите панель с результатами учеников, сгруппированными по группам

## Проверка сохранения данных

1. Зарегистрируйтесь как ученик
2. Пройдите тест
3. Закройте браузер
4. Откройте снова http://localhost:8000
5. Вы должны автоматически войти в систему
6. Ваши результаты должны быть сохранены

## Проверка выхода

1. Войдите в систему
2. Нажмите кнопку "Выйти" в правом верхнем углу
3. Вы вернётесь на экран входа
4. Данные сессии должны быть очищены

## Известные особенности:

- Ученики не требуют пароля (только имя, фамилия, группа)
- Учителя входят по логину и паролю
- Администратор входит по секретному ключу
- Все данные хранятся локально в браузере
- При регистрации ученика имя автоматически подставляется в форму прохождения теста
